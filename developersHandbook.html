1. Project Setup

Follow the steps below to set up the Playwright framework with TypeScript:

  1. Initialize a new Node.js project:
    2. npm init -y
  2. Install Playwright and other necessary dependencies:
    3. npm install playwright @playwright/test cucumber ts-node typescript allure-commandline --save-dev

2. Useful Commands

  • Run Tests: 
    npx playwright test
  • Install Playwright Browsers: 
    npx playwright install
  • Generate Allure Report: 
    allure generate allure-results --clean -o allure-report
  • Open Allure Report: 
    allure open allure-report

3. Generating Allure Reports

After running your tests, you can generate an Allure report using the following commands:

  1. Run the tests to generate Allure results:
    2. npx playwright test
  2. Generate the Allure report:
    3. allure generate allure-results --clean -o allure-report
  3. Open the Allure report:
    4. allure open allure-report

4. Contributing

If you wish to contribute to this framework, please follow these guidelines:

  • Fork the repository.
  • Create a new branch for your feature or bug fix.
  • Make your changes and commit them.
  • Push your changes to your forked repository.
  • Submit a pull request.
